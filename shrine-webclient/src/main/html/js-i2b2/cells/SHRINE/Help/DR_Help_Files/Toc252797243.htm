Semantics Preservation

Each participating institution's data is loaded with native coding schemes used by local information technology solutions. To understand SHRINE queries issued using terms from the Core ontology, a mapping is created and maintained for each institution that enables a translation from SHRINE Core ontology term to a corresponding local term, enabling the capability to perform a query at that institution.
