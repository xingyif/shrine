Site to RxNorm Associations Type Mapping

For the ingredient name (IN) and trade name (BN), the BN is derived from FDA Trade Name, and is the same as IN in most cases. In cases where they differ, such as extended release variants with something similar to XL in the trade name, that component of the IN is not in the BN.

A diagram indicates the datatype relationships for medication data.

This relationship map is described as follows:

  • IN → has_tradename → BN
  • IN → ingedient_of → SCDC
  • SCDC → has_tradename → SBDC
  • IN → ingredient_of → SCDF
  • SCDF → has_tradename → SBDF
  • SCDF → inverse_isa → SCD
  • SCD → has_tradename → SBD
