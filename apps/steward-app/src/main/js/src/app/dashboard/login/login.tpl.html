The role of the Data Steward is to ensure scientific and institutional integrity when accessing clinical data from SHRINE and its participating organizations. The Data Steward will:

  • Review and either approve or deny your Query Topic request
  • Audit your individual queris for comliance with the states Query Topic

Each project will require a separate Query Topic once your request has been approved by the Data Steward. you can start querying SHRINE The review cyle for straightforward query topics should be complete in two working days.
